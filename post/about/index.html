关于我

· 2019-01-25 ·

嗨 ~ 我叫胡青家，来自南京的 UI 设计师。专注于用户体验和交互设计。🤝

15 年初从机电自学转行 UI 以来，已在设计这行耕耘 6 年多，多年的工作实践让我从一个入行的菜鸟到现在能独立负责整个产品的 UI 设计工作，感谢自己辛勤付出和过往公司给予的支持。

目前主要负责政府医疗相关的 B 端产品设计，政府产品的设计不同于 C 端产品，可以近距离感受用户，B 端产品解决的是政府企业的具体问题，以可以完整的实现目标功能为核心，往往要牺牲用户体验去适应功能，而不会为了体验去删改功能，这就导致 B 端设计上更讲究交互上的清晰明确，视觉上也更为简洁、克制和干练。通过这么多项目的实践，目前对 B 端设计已经较为熟悉，能一次次为政府企业解决具体问题感到高兴。

从业这些年让我感受最深到一点是我们的世界瞬息万变，设计带给我的不仅仅是一份工作，更多的是通过这份工作去不断的学习，提高自己的知识和认知水平。
